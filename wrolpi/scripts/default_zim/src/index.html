This is the default WROLPi Zim

You are seeing this page because your WROLPi has no Zim files yet (or the Zim files it found could not be read). This built-in placeholder keeps the Kiwix viewer running until you download real content.

To add encyclopedias and reference libraries, open Zim in the WROLPi menu and download what you want — Wikipedia, Wiktionary, StackExchange, iFixit, medical and survival references, and more.

Downloaded Zim files are stored in your Zim directory (by default zims inside your media directory). Once at least one working Zim file is present, Kiwix will serve it here automatically and this page will disappear.
